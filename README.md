# Nidhi Agrawal Portfolio

Professional portfolio of **Nidhi Agrawal** — Senior Backend Engineer & Tech Lead with 5.5+ years of experience designing and operating production systems for enterprise hiring platforms. This site showcases experience, projects, skills, and contact information in a modern, recruiter-friendly format.

---

## Tech Stack

- **Next.js** – React framework with App Router
- **TailwindCSS** – Utility-first styling and responsive design
- **Framer Motion** – Animations and scroll-based interactions
- **TypeScript** – Type-safe JavaScript

## Run Locally

1. Clone the repository and install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## Build & Deploy

**Build for production:**

```bash
npm run build
```

**Run production build locally:**

```bash
npm run build && npm start
```

**Deploy (e.g. Vercel):**

- Connect the repo to Vercel (or your preferred host).
- Use the same build command: `npm run build`.
- No extra environment variables required for the static portfolio.

---

## Project Structure

- `app/` – Next.js App Router pages and layout
- `app/components/` – Reusable UI components (Header, Card, ThemeToggle, etc.)
- `app/globals.css` – Global styles and Tailwind
- `public/` – Static assets (resume, profile image)
